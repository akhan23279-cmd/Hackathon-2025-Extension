ClearView is a Google Chrome extension designed to improve web accessibility for all types of disabilites, including color blindness, dyslexia, visual impairments and motor disabilities.
Features: 
Color Fixer- provides a toggle to allow user to change text, link and background color
Text Formatter- provides a toggle to allow users to change font, text size, link size, line height and letter spacing
Screen Reader- provides a screen reader that will read the text on page to user
Voice Navigation- Supports the following commands: "Scroll Down", "Scroll Up", "Scroll to Top", "Scroll to Bottom", "Zoom In", "Zoom Out", "Start Reading", "Stop Reading", "Continue Reading", "Pause Listening", "Set text color to", "Set background color to", "Increase font size."
